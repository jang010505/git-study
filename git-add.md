## <code>git add "Filename"</code>란?
commit하기 전 저장소에 저장하는 명령어

### 사용 예시
<code>git add "filename"</code> : filename을 staging area에 저장</br>
<code>git add -A</code> 또는 <code>git add .</code> : working directory에서 수정된 파일git 을 staging area에 저장</br>
<code>git add -p</code> 또는 <code>git add --patch</code> : 현재 변경된 코드들의 파일들을 하나씩 보여주고, 해당 파일을 staging area에 저장 할 것인지 물음</br>
